# Note Space

A modern SaaS-style landing page for a productivity and collaboration platform focused on note-taking, classroom management, and team organization.

The project was built using pure HTML, CSS, and JavaScript while following responsive design principles and clean front-end development practices.

## ✨ Features

- 📱 Fully responsive design across desktop, tablet, and mobile devices
- 🎨 Modern SaaS-inspired UI/UX
- ⚡ Interactive front-end interactions using JavaScript
- 🧩 Clean and organized component-based page sections
